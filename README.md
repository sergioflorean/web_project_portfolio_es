# Portafolio

Personal Portfolio Website

This project is a personal portfolio website created to showcase web development skills, projects, and technical experience. The site introduces the developer, highlights selected projects, and provides ways for visitors to view work and get in contact. It is designed to present information in a clear, organized, and visually appealing way.

The website was built using HTML5 and CSS3, following semantic structure and modern layout practices. Flexbox and CSS Grid were used to create responsive layouts, while media queries and flexible units ensure the design adapts to different screen sizes. The project also uses Git and GitHub for version control and project management.
